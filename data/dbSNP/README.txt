rsID;location=genesymbol;option

option can be:
NON_CODING = SNP in non coding area
CODING_SYNONYMOUS = SNP in coding area, synonymous impact
CODING_MISSENSE = SNP in coding area, non-syn. impact, missense impact
CODING_NONSENSE = SNP in coding area, non-syn. impact, nonsense impact

cleans the data and gives you the right format for using it in this tool.
